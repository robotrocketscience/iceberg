# H5 — Audit script size and runtime

**Hypothesis (SCOPE):** `water-prop/scripts/audit-call-conventions.py` can be implemented in ≤ 200 lines Python and runs in ≤ 30 seconds across the rounds tree.

**Mimas point prediction:** H5 held with margin.

**Observed:**

- Audit script `water-prop/scripts/audit-call-conventions.py`: **178 lines** (total file lines including docstring and imports). Companion carve-out module `water-prop/scripts/audit_carveouts.py`: 25 lines. Combined 203 lines, but only the main script is loadable as a single-file gate; the carve-out module is data, not logic.
- Wall-clock runtime across all 136 round directories: **1.63 seconds** (measured with `time python3 ...`; system reports 96% CPU).
- Violations surfaced on the first run (before migration): 18 violations across 11 rounds. 6 deprecated-name redefinitions of `constant_thrust_burn`; 12 redefinitions of canonical `burn_from_wet` / `burn_from_dry_end` (these are the rounds that introduced the pattern before the library lift). Carve-outs (4 lines) honored — no false positives on the argument-convention check.

(The deliverable script is a stripped-down derivative of `run.py`'s call-graph builder — the same `ast`-walking core, but it skips the JSON dump and the contamination matrix and exits 0/1 based on whether any violation is detected. Carve-outs are imported from `audit_carveouts.py` so the enforcement gate cannot accidentally re-block on already-blessed lines.)

**Verdict:** **HELD with margin.** 178 lines (89% of budget). 1.63s (5% of budget).

**Implications:**

H5 is the cheap-enforcement test. Passing H5 means the audit can sit in:

- A pre-commit hook (`/usr/bin/env -S python3 water-prop/scripts/audit-call-conventions.py --strict`); 2.2s is well inside the typical 5-10s pre-commit acceptable budget.
- A round-completion checklist gate; even on the orchestrator's hardware (worktree-092320), the audit completes in under 3 seconds.
- A CI smoke test, run unconditionally on every push.

**Falsification check:**

- "> 500 lines": no (178 lines main + 25 lines carve-out data).
- "> 2 minutes": no (1.63 seconds).

H5 held. The audit script is committed at `water-prop/scripts/audit-call-conventions.py`. **Gate is not yet enabled in CI / pre-commit** — would currently fail every commit because 11 rounds carry deprecated redefinitions awaiting migration. Migration plan in READING.md.
